# 1F98A (codename Lishex)
A chess-engine built in C++.

### Features 

- iterative deepening Negamax framework
- Alpha-beta pruning 
- Square-centered 8x8 board representation 
- Piece-square tables for evaluation
- Universal Chess Interface (UCI) compatible
